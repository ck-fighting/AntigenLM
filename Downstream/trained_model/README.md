# Downstream Trained Models

The trained checkpoints for the three downstream tasks have already been prepared and are hosted on Hugging Face:

https://huggingface.co/cckai2017/Downstream_trained_model/tree/main

This repository includes trained models for:

- protective antigen prediction
- pMHC recognition
- pTCR recognition

Please download the required checkpoint files from the Hugging Face repository and place them under `Downstream/trained_model/` using the same directory structure expected by the scripts.
